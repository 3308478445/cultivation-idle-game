# 修仙放置游戏 v0.4 更新说明

## 🏆 新增成就系统

### 10个成就分5类

| 类别 | 成就名 | 解锁条件 | 奖励 |
|------|--------|----------|------|
| 📈 修炼 | 初窥门径 | 累计修炼100修为 | 50灵石 |
| 📈 修炼 | 小有所成 | 累计修炼10,000修为 | 100灵石 |
| 📈 修炼 | 登堂入室 | 累计修炼1,000,000修为 | 200灵石 |
| 🔮 境界 | 筑基成功 | 突破到筑基境 | 150灵石 |
| 🔮 境界 | 金丹初成 | 突破到金丹境 | 300灵石 |
| 💊 丹药 | 初识丹道 | 使用10颗丹药 | 1颗筑基丹 |
| 💊 丹药 | 炼丹大师 | 使用50颗丹药 | 2颗筑基丹 |
| 📜 奇遇 | 机缘巧合 | 触发10次奇遇 | 80灵石 |
| 📜 奇遇 | 仙缘深厚 | 触发50次奇遇 | 150灵石 |
| 💰 财富 | 腰缠万贯 | 累计获得1,000灵石 | 100灵石 |
| 💰 财富 | 富甲一方 | 累计获得10,000灵石 | 200灵石 |

### 新增功能

- **成就标签页** 🏆 - 在底部标签栏新增成就入口
- **成就通知动画** - 解锁成就时右上角显示金色通知动画
- **成就进度追踪** - 修炼、使用丹药、获得灵石、触发奇遇时自动追踪
- **成就分类展示** - 显示各类别解锁进度

### 技术实现

- 新增 `ACHIEVEMENTS` 常量数组定义所有成就
- 新增 `playerData.stats` 字段追踪：totalCultivation、pillsUsed、totalSpiritStones、totalEncounters
- 新增函数：`checkAchievements()`、`applyAchievementReward()`、`showAchievementNotification()`、`renderAchievements()`
- 在修炼循环、丹药使用、奇遇触发、突破成功、商店购买等位置集成成就检查
