# 修仙放置游戏 v0.5 更新说明

## 🏛️ 新增宗门系统

### 宗门功能

| 功能 | 说明 |
|------|------|
| 宗门列表 | 6个预设宗门可供加入 |
| 创建宗门 | 消耗5000灵石创建自定义宗门 |
| 加入宗门 | 立即成为宗门成员 |
| 离开宗门 | 可随时离开当前宗门 |

### 预设宗门

| 宗门名 | 等级 | 成员数 | 描述 |
|--------|------|--------|------|
| 太虚仙宗 | Lv.5 | 128 | 传承万年的正道巨擘 |
| 天魔宫 | Lv.4 | 89 | 魔道第一势力 |
| 青云门 | Lv.3 | 256 | 中土第一仙门 |
| 万剑宗 | Lv.3 | 67 | 剑道宗门 |
| 百花谷 | Lv.2 | 45 | 专注丹药与灵植 |
| 散修联盟 | Lv.1 | 1024 | 散修互助组织 |

### 每日任务

| 任务 | 目标 | 奖励 |
|------|------|------|
| 修炼 | 修炼1000修为 | 50贡献 + 30灵石 |
| 突破 | 完成1次突破 | 100贡献 + 50灵石 |
| 奇遇 | 触发2次奇遇 | 30贡献 + 20灵石 |
| 炼丹 | 使用3颗丹药 | 40贡献 + 25灵石 |

### 技术实现

- 新增 `PREDEFINED_SECTS` 常量定义预设宗门
- 新增 `SECT_SHOP_ITEMS` 常量定义宗门商店
- 新增 `DAILY_TASKS` 常量定义每日任务
- 新增 `playerData.sectId`、`sectContrib`、`sectDailyTasks` 等字段
- 新增函数：`renderSect()`、`joinSect()`、`createSect()`、`leaveSect()`
- 新增函数：`renderSectShop()`、`buySectItem()`、`renderDailyTasks()`
- 新增函数：`claimDailyTask()`、`updateDailyTaskProgress()`
