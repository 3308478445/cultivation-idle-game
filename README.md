# 修仙放置游戏

一款基于 HTML5 的修仙题材放置类游戏。

## 游戏特色

- 🎮 **放置玩法**：自动修炼，离线收益
- ⚔️ **境界系统**：练气 → 筑基 → 金丹 → 元婴 → 化神
- 📜 **功法系统**：习得功法，提升修炼速度
- 💊 **丹药系统**：服用丹药，获得增益效果
- 🏪 **商店系统**：购买功法和丹药

## 如何游玩

1. 点击"修炼"获得修为
2. 修为达到要求后点击"突破"
3. 突破成功进入下一境界
4. 购买功法和丹药提升实力
5. 继续修炼，最终化神飞升！
